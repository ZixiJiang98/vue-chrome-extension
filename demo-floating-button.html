Vibrant Wellness Floating Button Demo

This page demonstrates the floating button that will appear on web pages when the extension popup is not active.

Features:

  • Circular Design: 58x58px circular button with gradient background matching the Figma design
  • Vibrant Logo: Features the Vibrant Wellness logo with gradient colors
  • Hover Effects: Scales up slightly on hover with enhanced shadow
  • Pulse Animation: Pulses every 10 seconds to draw attention
  • Smart Visibility: Only appears when the extension popup is closed
  • Click Action: Opens the extension popup when clicked

Demo Controls:

The floating button is positioned in the bottom-right corner of the screen and will appear on all web pages when the extension is installed.
